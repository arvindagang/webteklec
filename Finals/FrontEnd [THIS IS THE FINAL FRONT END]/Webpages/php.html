Is a server-side scripting language. It enables the creation of data collection forms, generation of dynamic page contents, transfer of cookies and so much more. Php has the ability to output images, pdf files, flash movies and output any text like XHTML and other XML files. Php can autogenerate these kinds of files and store them in the file system. Moreover, PHP supports a wide range of databases it can produce its own database or import from others such as MySQL.
