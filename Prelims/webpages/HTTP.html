HyperText Transfer Protocol

What is HTTP?

 HTTP or Hypertext Transfer Protocol is an application-level protocol for distributed, collaborative, hypermedia information systems. It has been used by the World Wide Web for global information since 1990.
Terminologies

➧Connection
 Transport layer virtual circuit established between two application for the purpose of communication ➧Message
 Basic unit of HTTP communication.
➧Request
 HTTP request message
➧Response
 HTTP response message
➧Entity
 A particular representation of data resources, or reply from a service resource, that may be enclosed within a request or response message. Consists of metainformation in the form of entity headers and content in the form of an entity body.

Continuation

➧Client
 An application program that establishes a connection for the purpose of sending requests.
➧User Agent
 The client which initiates a request
➧Server
 An application program that accepts connections in order to service requests by sending back responses ➧Cache
 A program's local store of response messages and the subsystem that controls its message storage, retrieval and deletion. It reduces the response time and network bandwidth consumption on future same requests.
