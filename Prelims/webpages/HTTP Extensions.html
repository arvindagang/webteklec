HTTP Extensions

HTTP Extensions

 HTTP extensions are mechanism for handling public to client specification agreement between application of https in client, servers etc.

 HTTP Headers

  HTTP headers are additional information which are passed from the server to client. Mostly they are meta language responsible for the control of data sent by the server.

  Headers are grouped by their contexts.
  General Header - applies request and response, no relation with content data.
  Request Header - information of the resource(client) to be retrieve.
  Response Header - infromation of the resource(server) to be sent back.
  Entity Header - information about the body or content.

  Headers:
  Authentication - access and authorization.
  Caching - data compression and cache-control.
  Client hints - client options such as downloads.
  Conditional - used as a validator for resources retrieving or editing.
  Connection Management - management of connection.
  Content-Negotioation - documentary of the content or format of the body.
  Controls - server handling.
